# Team Agent Guide

Always write tests first. Prefer small, reviewable commits.

This file is the project's shared agent guidance. agent-sync renders it into
each target tool's native location:

- Claude Code → a managed section in `CLAUDE.md`
- Cursor and Codex → a managed section in `AGENTS.md`

Everything outside the agent-sync managed section in those generated files is
preserved across syncs — only the marked section is owned by agent-sync.
